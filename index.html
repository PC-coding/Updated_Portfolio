Nice to meet you!

Hey!
I'm a CUNY Queens College graduate who has found a passion in programming. I had my first experience with a computer in elementary school where I learned about Microsoft Word, Powerpoint and the glorious Internet at 7 years old. Our final project for our computer class in second grade was to make a customized blogsite. Ever since that cool interactive experience of building and designing my first 'website', I wondered how this data was changed and played with in the background to give us this cool user experience.

This brings us to post-college where I finally pursued my passion for computer software during the Coronavirus Pandemic and attended the NYC immersive Python Full Stack Software Engineering Program at Byte Academy. Byte taught me computer science fundamentals in Python with emphasis on data structures and algorithms in the first phase of their program. After passing the Python assessment, I moved onto phase two of the program where I learned more about web development in JavaScript, React.js, HTML, CSS, SQL and Flask with emphasis on API integration, using databases and building full stack applications with React.js + Flask.

After graduating Byte Academy, I've been working on my own personal projects that I've been interested in creating since the start of the program. You can see the rest of my projects and connect with me on Github or LinkedIn! :)
